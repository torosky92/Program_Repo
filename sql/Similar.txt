Carpetas
- Settings Programa que contiene las direcciones de las SQL y variables del sistema
- SettingsUser Programa de variable de usuario
- lib
	- CBD.db Base de datos de materia prima
	Informacion de cada materia prima retirada una tabla para cada presentacion y proveedor ( 4 listas) datos de materia prima retirado,
	montaje retorcido, parado de proceso, parado en montaje, peso, material final de retorcido
	- CBDR.db Comparacion de Base de datos de materia en proceso de retorcido
	- BDR.db Base de datos de materia en proceso de retorcido
	- PBDR.db Base de datos de materia en proceso de retorcido para pruebas
	- RMAQ.db Referencias de codigos, maquinas y paros de operacion
	- OP.db Base de datos de operarios
	- PW.db Autorizacion, Token y password
	- Inf.db Base de datos de las imagenes de ayuda para la interfaz de usuario
